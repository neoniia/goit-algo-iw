# Індивідуальна робота (goit-algo-iw)

Цей репозиторій містить розв'язання індивідуальної роботи з алгоритмів.

## Завдання 1. Жадібний алгоритм (розклад занять)

Файл: `task1.py`

**Запуск:**

```bash
python task1.py
```

Або можна запустити обидва завдання одним файлом:

```bash
run_all.bat
```

Програма:
- створює список викладачів з їхніми предметами;
- за допомогою жадібного алгоритму підбирає мінімальний набір викладачів, щоб покрити всі предмети;
- виводить розклад: який викладач які предмети викладає.

## Завдання 2. Локальний пошук (функція Сфери)

Файл: `task2.py`

Реалізовано три алгоритми локальної оптимізації:
- Hill Climbing
- Random Local Search
- Simulated Annealing

**Запуск:**

```bash
python task2.py
```

Програма виводить знайдені розв'язки (координати точки) та значення функції Сфери в цих точках для кожного з трьох алгоритмів.
